# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 2.x     | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public issue
2. Contact the maintainer via GitHub: [@valinerosgordov](https://github.com/valinerosgordov)
3. Provide a detailed description and steps to reproduce

You can expect a response within 48 hours. Valid reports will be addressed in a patch release.

## Scope

Daily Planner is a fully local desktop application. All data is stored on your machine at `%LOCALAPPDATA%\DailyPlanner\planner.db`. No data is sent to external servers (except checking for updates via GitHub Releases).
